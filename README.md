# lzs_decompress
Based on sosfomt_lzs.bms and quickbms source code modifications, tested with PSV game UPPERS, can decompress normally.
基于sosfomt_lzs.bms和quickbms源码修改而来,已测试psv游戏欲望之拳,解压正常

usage用法: 
lzs input_lzs_path  

example示例:
lzs F:\xunleidownload   

It will batch extract all lzs files in this path.
它将会遍历路径下的所有lzs文件进行解压。
